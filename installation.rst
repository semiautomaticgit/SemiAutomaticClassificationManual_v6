*********************
Plugin Installation
*********************

.. |br| raw:: html

	<br />

The Semi-Automatic Classification Plugin requires the installation of `GDAL, OGR <http://www.gdal.org/>`_, `NumPy <http://www.numpy.org/>`_, `SciPy <http://www.scipy.org/>`_ and `Matplotlib <http://matplotlib.org/>`_ (already bundled with QGIS).
|br|
This chapter describes the installation of the Semi-Automatic Classification Plugin for the supported Operating Systems.
